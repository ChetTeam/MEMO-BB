# GTA5RP BLACKBERRY MEMO LAWS
![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)
![Platform](https://img.shields.io/badge/platform-Windows-lightgrey.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

Автономная памятка для быстрого доступа к [ЗБ](https://forum.gta5rp.com/forums/zakonodatelnaja-baza.262/) и [Правилам](https://forum.gta5rp.com/categories/pravila-servera-blackberry.224/) Сервера [Blackberry](https://forum.gta5rp.com/#igrovye-servera.13) Проекта [GTA5RP](https://gta5rp.com/). Продукт спроектирован для Сотрудников Гос. Структур: минимизирует alt-tab, ускоряет процесс ориентировки по форуму и обеспечивает актуальную информацию.

## ⚙️ Ключевой функционал
* **Глобальный вызов:** Мгновенный рендер интерфейса поверх всех окон по нажатию `Alt + E`.
* **Автоматизированность:** Встроенное ядро на Puppeteer вытягивает актуальные законы напрямую с форума.
* **Приятный Стиль:** Отзывчивый интерфейс с мгновенным переключением между Общим форумом и Памяткой без перезагрузки Программы.
* **Smart Copy:** Интерактивные бейджи для копирования конкретных частей статей в буфер обмена в один клик (формат: `Ч. 1 48 АК`).

## 📥 Установка
1. Перейдите в раздел [Releases](../../releases).
2. Скачайте актуальный установщик `GTA5RP MEMO LAWS Setup.exe`.
3. Установите приложение (ярлык будет создан автоматически).
4. Запустите клиент и вызывайте окно в игре через `Alt + E`.

*Перед установкой обязательно ознакомьтесь с [Условиями использования](TERMS.md).*

## 📞 Обратная связь и Уведомление об ошибках
**🔴 Внимание: Проверка корректности наказаний**
> В алгоритме автоматического извлечения наказаний (штрафы/аресты) из текста кодексов могут возникать синтаксические сбои. Как обнаружить баг: Наказание выводится в интерфейсе в двух местах — в строке заголовка статьи (справа) и непосредственно под текстом самой статьи. Если вы заметили визуальное несоответствие данных между этими двумя точками, значит информация справа некорректная. О подобных несоответствиях, багах парсинга или ошибках интерфейса незамедлительно сообщайте разработчикам для выпуска патча.
