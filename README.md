# GTA5RP BLACKBERRY MEMO LAWS
![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)
![Platform](https://img.shields.io/badge/platform-Windows-lightgrey.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

Автономная памятка для быстрого доступа к ЗБ и Правилам Сервера [Blackberry](https://forum.gta5rp.com/forums/server-no4-blackberry.223/) Проекта [GTA5RP](https://gta5rp.com/). Продукт спроектирован для Сотрудников Гос. Структур: минимизирует alt-tab, ускоряет процесс ориентировки по форуму и обеспечивает актуальную информацию.

## ⚙️ Ключевой функционал
* **Глобальный вызов:** Мгновенный рендер интерфейса поверх всех окон по нажатию `Alt + E`.
* **Автоматизированный парсинг:** Встроенное ядро на Puppeteer вытягивает актуальные законы напрямую с форума, игнорируя базовые ограничения Cloudflare.
* **SPA-архитектура:** Отзывчивый интерфейс с мгновенным переключением между Общим форумом и шпаргалкой (Мини-форумом) без перезагрузки DOM.
* **Калькуляция наказаний:** Регулярные выражения автоматически извлекают и привязывают суммы штрафов и сроки ареста к статьям АК, УК и ДК.
* **Smart Copy:** Интерактивные бейджи для копирования конкретных частей статей в буфер обмена в один клик (формат: `Ч. 1 48 АК`).

## 📥 Установка
1. Перейдите в раздел [Releases](../../releases).
2. Скачайте актуальный установщик `GTA5RP MEMO LAWS Setup.exe`.
3. Установите приложение (ярлык будет создан автоматически).
4. Запустите клиент и вызывайте окно в игре через `Alt + E`.
